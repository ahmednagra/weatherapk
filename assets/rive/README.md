# Rive weather animations

Drop community-sourced, license-vetted `.riv` weather animations here.

- File expected by `RiveWeather` (in `core/widgets/app_widgets.dart`): `weather.riv`
- State machine name + input names are mapped in `RiveWeather` — adjust the
  mapper there to match the pack's actual state machine / inputs.
- Source + license of each pack MUST be recorded below.

If `weather.riv` is absent or incompatible, the UI falls back to the static
grayscale-free webp hero in `assets/images/wx/` — so this is non-blocking.
